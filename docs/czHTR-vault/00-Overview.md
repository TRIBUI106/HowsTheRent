# Project Overview — czHTR

## Current shape
- Brownfield-light project: `htr-backend` is a minimal Spring Boot skeleton.
- Planning source: `master-plan.md` describes a full property management system.

## Intended stack from plan
- Backend: Java 21, Spring Boot, Maven, PostgreSQL, Redis, Quartz, JWT, PayOS, MinIO.
- Frontend: React 18, TypeScript, Vite, Tailwind, shadcn/ui, TanStack Query.

## Key source files
- `master-plan.md` — product/database/API implementation plan.
- `htr-backend/pom.xml` — minimal Maven project.
- `htr-backend/src/main/java/chez1s/htrbackend/HtrBackendApplication.java` — Spring Boot entrypoint.
